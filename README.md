# W3School Practice Repository

This repository contains my C++ practice programs for learning fundamental concepts such as arrays, conditions, loops, and functions.  
It serves as a collection of exercises and examples to strengthen my understanding of C++ basics.

## Setup Instructions
To run any program in this repository:

1. Clone the repository:
   ```bash
   git clone https://github.com/rish3119/W3School.git
   cd W3School
   ```
2. Compile the desired file:
   ```bash
   g++ filename.cpp -o output
   ```
3. Run the program:
   ```bash
   ./output
   ```

## Usage Example
Here’s an example using an array summation program:

Input:
```
5
10 20 30 40 50
```

Output:
```
Sum: 150
```

## Requirements
- g++ compiler (or any C++17+ compatible compiler)
- Runs on Windows, Linux, or macOS
- Basic knowledge of C++ syntax
